SVG Icons

Proof of concept for the consideration of PixelBuddha

Responsive test

Each icon is created by using an svg tag with the class of "icon". Inside of that, the user can refer to a single-word id for the icon.

<svg class="icon">
  <use xlink:href="#backpack"></use>
</svg>

These are actually symbol references to the main library of SVGs, which can be included in the HTML.

The main benefits of doing it this way are outlined here: CSS Tricks Article
